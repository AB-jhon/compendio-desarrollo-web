# 🚀 Compendio de Desarrollo Web y Diseño de Interfaz

> **Proyecto de Titulación: Especialidad en Diseño Web**
> **Institución:** Centro Universitario San Ángel (CUSA)
> **Periodo Académico:** 2016 - 2019 (Entrega 2026)
> **Autor:** Jhonathan Anota Badillo

Este repositorio constituye la evidencia técnica y práctica del aprendizaje adquirido durante la especialidad. Cada módulo del código ha sido diseñado para validar los fundamentos teóricos y metodológicos descritos en el compendio escrito, integrando estándares modernos (2024-2026).

¡Bienvenido a mi proyecto final! Esta es una Single Page Application (SPA) profesional que centraliza conceptos avanzados de desarrollo moderno, con una arquitectura de componentes escalable y un diseño de experiencia de usuario (UX) de alto nivel.

---

### 🛠️ Stack Tecnológico
| Capa | Tecnología |
| :--- | :--- |
| **Frontend** | React 19 + Vite ⚡ |
| **Estilos** | Tailwind CSS (Glassmorphism & Dark Mode) 🎨 |
| **Iconos** | Lucide-React ✨ |
| **Animaciones** | CSS Keyframes (Fade-in Transitions) 🎬 |

---

### 🌟 Características Principales
* **Diseño Responsivo:** Adaptado para móviles, tablets y escritorio.
* **Modo Oscuro Dinámico:** Switch de estado con persistencia visual.
* **Navegación Fluida:** Transiciones suaves entre secciones sin recarga de página.
* **Accesibilidad e Inclusión:** Implementación basada en las pautas WCAG 2.1 para garantizar un diseño para todos los usuarios.

### 📚 Secciones Educativas (Mapeo de los 8 Temas) 
    * `Intro-Web`: Fundamentos, historia y evolución ética de la Web (Tema 1).
    * `Programacion`: Estándares, frameworks modernos y editores interactivos (Temas 2 y 3).
    * `BasesDatos`: Comparativa visual entre paradigmas SQL y NoSQL (Tema 6).
    * `Proyectos`: Gestión profesional mediante tableros Kanban, principios Ágiles y flujos CI/CD (Tema 8).
